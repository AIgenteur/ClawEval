Q: Which operating systems and mobile devices are supported by CloudSync Pro?
A: The service officially supports Windows, Mac, and Linux desktops. Currently, Android support is not available, but the phrasing implies it is a future release rather than a permanent limitation.

This implies the 30-day rule applies to Free and Pro users, while Enterprise gets an extended 90-day window.

This means the 99.9% guarantee applies exclusively to Pro and Enterprise users, while Free users operate without a formal SLA guarantee despite the general statement.

Q: Can I upgrade from the Free tier to the Pro tier?
A: Yes, users can upgrade from the Free tier to the Pro tier ($9.99/mo) to access unlimited storage (removing the 5GB cap) and the 10GB file size limit, as well as the 99.9% uptime SLA and the 30-day file recovery window.

Q: When will Android mobile support be available?

Q: Are there any differences in file encryption between tiers?
A: No, file encryption is consistent across all tiers. All files are encrypted with AES-256 both at rest and in transit, regardless of whether the user is on the Free, Pro, or Enterprise plan.
