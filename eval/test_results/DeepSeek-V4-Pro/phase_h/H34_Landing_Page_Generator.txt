        navLinks.querySelectorAll('a').forEach(link => {

                navLinks.classList.remove('open');
                mobileMenuBtn.setAttribute('aria-expanded', 'false');
            });

        // FAQ accordion
        document.querySelectorAll('.faq-question').forEach(button => {

                const faqItem = button.parentElement;
                const isOpen = faqItem.classList.contains('open');
                // Close all
                document.querySelectorAll('.faq-item').forEach(item => item.classList.remove(
                    'open'));
                document.querySelectorAll('.faq-question').forEach(btn => btn.setAttribute(
                    'aria-expanded', 'false'));
                // Open clicked (if wasn't already open)
                if (!isOpen) {
                    faqItem.classList.add('open');
                    button.setAttribute('aria-expanded', 'true');
                }
            });
        });

        // Header scroll shadow
        const header = document.getElementById('site-header');

                header.classList.remove('scrolled');

### Design & UX Highlights

The page uses soft gradients, rounded shapes, and clear content sections to create a modern and approachable SaaS product presentation around the early concept of "AI-enhanced productivity."

- **Visual Hierarchy:** A **bold hero section** with a glowing illustration and a clear CTA ("Start Free Trial") grabs attention first. The sticky navigation bar and smooth scroll links make it easy to explore.
- **Trust & Credibility:** Social proof ("Trusted by 10,000+ users"), a **before/after comparison** block, and detailed testimonials build confidence. The color coding (red/green) in the comparison makes the product's value instantly clear.
- **User-Focused Layout:** The **"How it Works"** section uses numbered cards, the pricing table highlights the most popular plan, and the accordion FAQ keeps the page clean while answering common questions.
- **Interactive Feedback:** Subtle hover effects on cards and buttons, a mobile menu toggle, and the FAQ accordion create a polished, responsive experience. The newsletter subscription form provides a clear path for engagement.

---

**Optimization Tip:** You can replace the placeholder logo names (TechCrunch, Forbes) and social media icons with your actual brand assets. The FAQ and testimonial content should also be updated to match your specific product details.
