**Finally an AI assistant that doesn't gaslight me about my own codebase**

Been using the beta for ~6 weeks on a 200k LOC TypeScript project. Posting because I was skeptical as hell after Copilot kept suggesting imports from files that don't exist.

The thing that actually works: it indexes your whole repo locally, so it knows when you renamed that function last month and where it's actually used. No more "this definitely works" hallucinations about APIs.

Downsides: indexing takes ~10 min on first load for big projects. RAM usage spikes during that. Also the "explain this" feature is sometimes too verbose—like junior dev documentation energy.

Not affiliated, just tired of tools made by people who don't ship production code. Happy to answer questions if anyone's curious.
