### Explanations:
1. **Python 3.0 was released in December 2008.**
   - **FALSE**: Python 3.0 was released in **February 2008** (not December).

2. **Git was initially released in 2004 by Linus Torvalds.**
   - **FALSE**: Git was released in **2008** (not 2004). Linus Torvalds was the primary developer.

3. **The first version of JavaScript was created in 10 days.**
   - **FALSE**: The first version of JavaScript (ECMAScript 1, or ECMAScript 262, 1st Edition) was finalized in **1997** (not created in 10 days). The initial implementation by Brendan Eich took about **96 hours** (roughly 4 days).

4. **Redis is single-threaded for all operations including I/O.**
   - **TRUE**: Redis is indeed single-threaded for all operations, including I/O, due to its event-driven architecture.

5. **Docker containers share the host OS kernel.**
   - **FALSE**: Docker containers run in **isolated user spaces** (using cgroups and namespaces) but do **not** share the host OS kernel. They are lightweight virtualization.

6. **REST was defined by Roy Fielding in his 2000 doctoral dissertation.**
   - **TRUE**: REST (Representational State Transfer) was formally defined in Fielding's PhD dissertation, *"Architectural Styles and the Design of Network-based Software Architectures"* (2000).

7. **SHA-256 has never had a collision found.**
   - **FALSE**: SHA-256 has been proven to have **collisions** (though not a practical collision for most use cases). The first collision was found in **2017** (though it required a very large dataset).

8. **IPv6 addresses are 64 bits long.**
   - **FALSE**: IPv6 addresses are **128 bits** long (not 64 bits). The first 64 bits represent the link-local address, and the remaining 64 bits are for global routing.

9. **The CAP theorem states you can only have 2 of: Consistency, Availability, Performance.**
   - **FALSE**: The CAP theorem states you can only have **two out of three** (Consistency, Availability, Partition tolerance), not just two. The third is **Partition tolerance**.

10. **Kubernetes was originally developed at Google based on their Borg system.**
    - **TRUE**: Kubernetes was derived from Google's Borg system, which was later open-sourced as Kubernetes.

11. **The first commit to Linux kernel was in 1991.**
    - **FALSE**: The first commit to the Linux kernel was in **1991**, but the first public release was **August 1991** (Linux 0.01). The first commit was made by **Linus Torvalds** on **August 25, 1991**.

12. **TCP guarantees in-order delivery of packets.**
    - **TRUE**: TCP is a reliable, connection-oriented protocol that ensures in-order delivery of packets.

13. **UTF-8 can encode any Unicode character in at most 4 bytes.**
    - **TRUE**: UTF-8 is a variable-width encoding that can represent any Unicode character in **1 to 4 bytes**.

14. **MongoDB is ACID compliant for single-document operations.**
    - **FALSE**: MongoDB is **not ACID compliant** for multi-document transactions (it supports ACID for single-document operations, but not for multi-document transactions).

15. **HTTP/2 was standardized in 2015.**
    - **TRUE**: HTTP/2 was standardized in **2015** (RFC 7540).

16. **Rust was first released by Mozilla in 2010.**
    - **FALSE**: Rust was developed by **Graydon Hoare** (not Mozilla) and first released in **2010** (not 2015). Mozilla adopted it later.

17. **WebAssembly became a W3C recommendation in December 2019.**
    - **TRUE**: WebAssembly (WASM) was a W3C recommendation in **December 2019**.

18. **The maximum length of a URL is 2,048 characters per the HTTP spec.**
    - **FALSE**: The maximum length of a URL is **2,083 characters** (not 2,048) per the HTTP spec (RFC 7230).

19. **Python's GIL prevents all multi-threaded execution.**
    - **FALSE**: The GIL prevents **true parallel execution** in Python threads, but it does not prevent **multi-threading** (which is still useful for I/O-bound tasks). It does not prevent all multi-threaded execution.

20. **gzip compression uses the DEFLATE algorithm.**
    - **TRUE**: gzip uses the **DEFLATE** algorithm (a combination of ZLIB compression and Huffman coding).
