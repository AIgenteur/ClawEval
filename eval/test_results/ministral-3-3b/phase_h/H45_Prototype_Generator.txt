                if max_len is not None and len(value) > max_len:

        elif field_type == "int":
            if not isinstance(value, int):

            else:
                min_val = schema.get("min")

                min_val = schema.get("min")

            if not isinstance(value, bool):

                errors.append(f"Field '{field_name}' must be dictionary")

This version properly tracks the field name throughout the validation process. The class will:

1. Check if all required fields are present
2. Validate the type of each field
3. For numbers, check min/max values if specified
4. For strings, check min/max length if specified
5. For lists, check min/max length if specified
6. Collect all errors and return them in the result

The errors will include specific information about which field failed validation and why.
